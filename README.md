# Recipes API
API for query recipes from third parties.

### Prerequisites

You need use PHP7.2.

### Installing

You need clone this project and execute:

```
composer install
```

Then update project:
```
composer update
```

### Execute local server
```
php -S 127.0.0.1:8000 -t public
```

### Example request
```
http://localhost/api/1/recipes?provider=RecipePuppy&query=steak&page=1
```
